linux快速搭建ss个人服务

Just Cost 3 mins

  • 1 下载代码
    • 方法1 适合安装了git客户端的服务器 
      git clone git@github.com:caroltc/shadowsocks-rm.git && sudo chmod +x shadowsocks-rm/*.sh
    • 方法2 
      wget https://github.com/caroltc/shadowsocks-rm/archive/master.zip -O shadowsocks-rm.zip && unzip shadowsocks-rm.zip && sudo chmod +x shadowsocks-rm/*.sh

  • 2 部署

    • 编辑配置config.json,确定要使用的端口和密码
    • 开启服务 sh start.sh, 通过ps aux | grep python 看到python server.py进程一般就成功了
    • 防火墙端口开放访问，可以运行我做好的脚本open_port.sh 端口号,如果不能用那就自己找下iptable的命令
  • 3 Join your life~
